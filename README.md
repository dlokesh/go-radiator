# go-radiator

Build radiator for [Thoughtworks Go][go]

## Installation

    $ brew install leiningen

## Usage

Checkout project and run:

    $ lein uberjar
    $ java -jar target/go-radiator-0.1.0-standalone.jar http://localhost:8153/go/cctray.xml
